# LINE 歷史對話記錄分析資料夾

## 用途
此資料夾用於存放歷史 LINE 對話記錄，供 AI 分析以優化客服 Bot。

## 支援格式
- CSV 檔案
- TXT 文字檔
- JSON 格式
- 截圖（PNG/JPG）

## 分析目標
1. **常見問題提取** - 找出客人最常問的問題
2. **回答模式優化** - 學習有效的回答方式
3. **語氣風格改進** - 調整 Bot 的語氣更貼近真人客服
4. **知識庫擴充** - 補充遺漏的 FAQ

## 使用說明
1. 將 LINE 對話記錄檔案放入此資料夾
2. 通知 AI 開始分析
3. AI 會生成分析報告和優化建議

## 注意事項
- 請確保對話記錄已去除個人敏感資訊（如身份證號、信用卡號等）
- 保留的資料：訂單編號、房型、日期、常見問題等
